# Euclidean
This is the rotationally invariant straight line (*bee* line) distance between two points.

### Parameters
This kernel does not have any parameters.

### Example
```php
use Rubix\ML\Kernels\Distance\Euclidean;

$kernel = new Euclidean();
```
